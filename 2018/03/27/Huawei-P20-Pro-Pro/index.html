Huawei P20 Pro

华为 P20 Pro 是中国制造商最新的旗舰型智能手机，它在摄像头技术方面可以说是我们近来看过的最具创新性的手机。快速查看这款手机的背板，即可发现它是第一款采用三摄像头设置的智能手机。但是，华为并非简单地将第三个传感器和镜头添加到目前的双摄像头系统上而已，这款新机型在几个方面也展现了卓尔不凡的性能：
  1. 主摄像头配备有一般手机没有的超大型1.78英寸图像传感器，其大小是三星 Galaxy S9 的1/2.55英寸芯片的2倍；即使f/1.8光圈相对较小，它可捕捉的光线量却比大多数竞争机型的小型芯片高出20％左右。同时黑白副摄像头也能提供大量的光信息来帮助主摄像头。
  2. 主摄像头的图像传感器使用Quad Bayer结构，总体像素为4000万像素。其输出数据分别由2 × 2像素的集合读取（bining）来实现，总共可输出1000万像素的图像。
  3. P20 Pro的远摄摄像头带有光学防抖系统，等效焦距为80毫米，其拍摄距离远大于最新的iPhone或三星 Galaxy的2倍远摄模块。由于它的主摄像头结合了一个2000万像素黑白副摄像头，可以在2倍变焦下拍出非常不错的细节，因此其工程师可以专注于为P20 Pro开发出具有更远摄距的远摄镜头。

摄像头主要规格：

  • 三摄像头设置
  • 主摄像头：4000万像素，1/1.73英寸RGB彩色图像传感器，具有Quad Bayer结构，f/1.8光圈镜头，等效焦距27毫米
  • 副摄像头：2000万像素，1/2.78英寸黑白图像传感器，f/1.6光圈镜头，等效焦距27毫米
  • 远摄摄像头：800万像素，1/4.4英寸RGB彩色图像传感器，f/2.4光圈镜头，等效焦距80毫米，配有光学防抖（OIS）
  • 主摄像头配有相位检测自动对焦（PDAF）
  • 4K 视频（非自动模式）
  • 960 fps，720P 慢动作视频
  • 6.1英寸，2240 x 1080 分辨率显示屏
  • 麒麟 970 芯片
  • 大光圈模式下的默认等效焦距: 55毫米

Huawei P20

新的华为 P20 配备了类似于之前的华为高端机型的双摄像头设置，而不是更大、更昂贵的姐妹机型 P20 Pro 的三摄像头、大型四色传感器和专用的远摄镜头。P20 主要的1/2.3英寸RGB彩色图像传感器搭配了一个1/2.78英寸黑白传感器，并将来自两个摄像头的图像数据以计算方式合并，以产生使背景模糊的散景效果，同时还可以进行降噪处理、增加图像细节和动态范围。
这两个模块都具有27毫米等效焦距，主摄像头配有f/1.8光圈镜头，黑白摄像头配有f/1.6光圈镜头。后者在计算式成像中融合多帧堆叠时特别有用。

摄像头主要规格：

  • 主摄像头：1200万像素，1/2.3英寸RGB彩色图像传感器，f/1.8光圈镜头，等效焦距27毫米
  • 副摄像头：2000万像素，1/2.78英寸黑白图像传感器，f/1.6光圈镜头，等效焦距27毫米
  • 相位检测自动对焦 （PDAF）
  • 5.8英寸，2240 x 1080 分辨率显示屏
  • 大光圈模式下的默认等效焦距: 55毫米
